# OS2021_HW1
## Module
a. Load the module my_info to the proc system by first compile
```sh
make
make ins
```
b. See the module info by the instruction
```sh
cat proc/my_info
```
c. When you want to unload the module, use this
```sh
make rm
make clean
```
## Application program
a. Compile the file by 
```sh
gcc app.c -o app
./app
```
